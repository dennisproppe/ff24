Create.xyz

Create.xyz is a powerful no-code platform that enables you to build web applications without writing code. Perfect for rapid prototyping and MVP development.

CAUTION: If you use the free version, your project will be shared with all other Create users. Don't do this if you want to test an idea where secrecy is key. For that, I'd recommend spending 19$ for the first month, and divide these costs in the group.

Building an AI Fitness Coach

Here's an example conversation showing how to build a muscle build-up AI coach with Create.xyz:

  1. User:

    Hi, I'd like to build a muscle build-up AI coach. Basically I would like to connect to a LLM, modify the system prompt so that it is very much focused on creating answers for questions regarding muscle build up for middle aged men.

  2. Assistant:

    You should add ChatGPT as an integration from the / Menu.

  3. Assistant:

    The next Step (Step 2 out of 2) is to add the Markdown Renderer from the / Menu, so that the answers are rendered as Markdown.

  4. User:

    Can you use this to render the AI answers as markdown?
