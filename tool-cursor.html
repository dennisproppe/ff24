About Cursor

Cursor is an innovative code editor that puts AI at the forefront of the development experience. It helps developers write, understand, and transform code more efficiently.

Key Features

  • AI-powered code completion
  • Natural language code generation
  • Code explanation and documentation
  • Smart code refactoring

How to Use

  1. Download Cursor from cursor.sh
  2. Install and set up your development environment
  3. Open your project in Cursor
  4. Start coding with AI assistance

Building a Plane Tracker Application

Here's an example conversation showing how to build a plane tracking application with Cursor:

  1. User:

    Hi, I'd like to start a new go project. Can you intialize it in this folder?

  2. User:

    Ok, cool, that worked. I'd now like to set up a small web application that allows you to Enter a plane type and then displays the airplanes that are in the air at that moment on a google map. For that we can use a very simple GET call to the API of @https://api.airplanes.live/v2/type/A388, where A388 is one possible type of airplane.

  3. User:

    Ok, great, the problem is, I dont have a Google Maps API key. I created a solution lately, where I did not have to supply one, is this also possible?

  4. User:

    Wow, awesome. I'd like to add a table under the map, where the retrieved JSON data from the API is listed with the elements that are in the Aircraft struct.

  5. User:

    Ok, now for the last step I'd really like to apply a nicer design to the website. Can you style it with Bootstrap?

  6. User:

    Thanks, that is cool. Can you adjust the color scheme and the spacing etc. from this page to this page? @https://future-founder.com/

This example demonstrates how Cursor can help you build a complete web application, from project initialization to styling, with natural language prompts.

Building a Typing Speed Test Application

Follow this conversation to build a typing speed test application with Cursor:

Prerequisites:

  • Open Cursor with a new folder in the appropriate place
  • Create a file: hello.go in this folder

  1. User:

    Can you build me a little console hello world example, but have it say "Hello, word!"?

  2. User:

    Can you change this go app to show hello, word in a web app instead of a console app?

  3. User:

    Hi, can you change to app so that it uses a standard bootstrap css design?

  4. User:

    Ok, that was great - Can we restructure the project a bit, so that the HTML part of the code resides in a separate file that is then referenced by the hello.go?

  5. User:

    Ok, now I'd like to add a little funtionality to the app. I'd like to have a menu bar on the left side, with two entries: home and typing speed, can you do that?

  6. User:

    Ok, now I would like to generate an actual typing speed test page. The page should have a text box in that I can type a text that has been displayed above the box. When I type in that box, the speed and accuracy of the typing should be measured and displayed after typing is finished.

  7. User:

    Ok, this works fine. It just has a small problem: Whenever i have a typo in the text, it never finishes, so I always have an accuracy of 100%. Can you change the logic so that it stops also when the test text length is reached?

  8. User:

    Almost done. Can you add some more text examples to type?

This example shows how to incrementally build a web application with Go, starting from a simple console app and gradually adding features and improvements.
